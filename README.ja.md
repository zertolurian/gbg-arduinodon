2021年のNintendo Switchゲーム「**ゲームビルダーガレージ**」（英語名：**GBG**、日本語名：**はじプロ**）向けに設計された、インタラクティブなHTMLユーザーインターフェースと**Arduino**ファームウェアです。このツールを使えば、はじプロ用のコード設計図をどのブラウザからでも簡単に作成・シミュレーション・共有でき、さらにArduinoを使って実際のゲーム内に自動で再描画することができます。（Switchやゲームの改造は一切不要！）

## ⚙️ 仕組み

「**ゲームビルダーガレージ**」の*プログラム画面*では、USBマウス操作がネイティブにサポートされています。

**Arduino**はプログラム可能なマイコンで、USBマウスとして認識できるモデルがいくつか存在します。

丁寧なスクリプティングと自動化によって、あらゆる設定のノードンをマウスの動きとクリックの長いリストに逆算することができます。このプロジェクトは、複数のノードンをはじプロ内で同じ構成として再現するためのマウスコマンドの一連の流れへ変換するプロセス全体を自動化します。

これはマウスコマンドを一方向に送信するだけなので、ユーザーが誤ってマウスを動かしてしまうなど、ゲーム内で予期しない出来事が起きた場合に対応することはできません。また、改造オブジェクトなど、プレイヤー自身が作れないものを作ることもできません。

## 🛠️ はじめかた

**[はじプロ向けArduinoツール全般に不慣れな方は、まずこちらの「GBG Arduino テクスチャツール」に関する動画をご覧になることをお勧めします。非常に似たプロセスです。（VideoDojo制作、スクリプト：BorriおよびScrubz）](https://youtu.be/PTDiTS-Exjk?si=7BVb8NnknOZa9iBn)**

以下の手順に従って、ブラウザからはじプロ本体へコードをインポートしましょう！

### 1. ArduiNodon UIで作成する
1. **[ライブWebUI](https://zertolurian.github.io/gbg-arduinodon/)** またはローカルの `ArduiNodon-UI.html` を対応ブラウザで開く。
2. ゼロからコードを作成するか、他の人の `.ndnx` 設計図をインポートする。
3. *（任意）* UI上ではじプロ内での動作をシミュレーションする！
4. コードを `.ndnx` ファイルにエクスポートして簡単に共有するか、*クリップボードにコピー* してArduinoスクリプトに貼り付け、はじプロへインポートする。

### 2. ハードウェアの準備
1. ArduinoボードをUSBケーブルでパソコンに接続する。**（このスクリプトはATmega32u4を搭載したArduino Leonardoなど、ネイティブでマウスエミュレーションをサポートするボードのみ対応しています。Arduino UNOは*非対応*です）**
2. *（任意）* GNDと8〜13番ピンのいずれかを繋ぐボタンを接続する。

### 3. Arduinoへアップロードする
1. Arduino IDEで `ArduiNodon-Importer.ino` ファイルを開く。
2. IDEのツールメニューから正しいボードとポートを選択する。
3. NicoHoodによる **HID-Project** ライブラリがインストールされていることを確認する。「ツール → ライブラリを管理 → "HID-Project"を検索 → インストール」の順に操作する（未インストールの場合）。
4. *（任意）* スクリプト上部のConfigにある設定（遅延時間、上書きモード、コードを実行するプログラム画面上の位置など）を好みに応じて変更する。
5. *（ボタンを接続していない場合）* Configから*ノーボタンモード*を有効にする。
6. ArduiNodon UIからエクスポートしたコードを、200行目付近の **NODON CODE SPACE** に貼り付ける（下にスクロールすればすぐわかります！）。
7. **アップロード**をクリックしてマイコンにコードを書き込む。コンソールにアップロード成功のメッセージが表示されたら、パソコンからArduinoを取り外す。

### 4. はじプロへインポートする
1. ArduinoをドックのUSB-Aポート経由でSwitchに接続する。（Switch Liteをお使いの場合はUSB-AからUSB-Cへの変換アダプターが必要で、SwitchのUSB-Cポートに直接接続します）
5. *（ボタンを接続していない場合）* 30秒（またはConfigに設定した遅延時間）待つと、スクリプトが自動的に開始されます。終了したら、再びスクリプトが動作しないうちにArduinoを取り外してください。
6. *（ボタンを接続した場合）* ボタンを押すと、スクリプトがすぐに開始されます。ミスが発生した場合は、ボタンを再度押して再実行できます。
7. スクリプトが終了するまで待ち、この間はコントロールを妨げないようにしてください。妨げた場合はスクリプトがミスを犯すため、再実行が必要になります。
8. 完成！はじプロにコードがインポートされました！🎉
