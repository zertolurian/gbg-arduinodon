ArduiNodon UI is a visual editor emulating Game Builder Garage, built for the ArduiNodon Arduino script. Place and connect Nodons just like in GBG, and then Export to share with others or paste into the ArduiNodon script to recreate all the Nodons and connections within GBG.

INFO
Simulate Mode emulates exactly what will happen within the Nodon grid when it is implemented in Game Builder Garage. Use the header controls to start, pause, and step through the simulation frame by frame. Click any Nodon on the grid to interact with it (for Input Nodons) or inspect its real-time inputs, outputs, and internal state.

In Normal View, the simulation will run exactly like in-game, displaying frame-by-frame at 60 frames per second by default.

In Subframe View, the simulation will display all the hidden subframes that the game calculates between each frame at 16 subframes per second by default.

Note: Just like in GBG, frame 0 only lasts 1 subframe, and several Nodon functionalities only become enabled at certain points throughout frames 1-3.

Changelog
V1.3 — 6/22/2026
Landscape mobile header, expanded multi-select capabilities, link sharing, and several bug fixes.
Sharing
Added "Share as Link" in the Export dialog which copies a URL with the grid encoded into it; opening that link automatically imports the grid
Multi-select
Duplicating Nodons which share connections will also duplicate their connections
Individual connections can now be selected by clicking them directly, or when a freeform-drawn loop encloses both ends of a connection; this will duplicate the connection for any newly duplicated Nodons
Move, Copy, and Delete buttons now appear directly on each multi-selected tile (the same buttons used for single selection), and clicking any of them runs the multi-version of that action
Loop detection slightly improved (also applied for Delete Mode)
Mobile
Header now adapts to landscape orientation of varying widths
Port long-press detection for making connections greatly improved
Build Mode
Sidebar footer (when a Nodon is selected) now include Move and Copy buttons as well
Selecting an empty tile now shows only Deselect in the footer
UI
Restyled the scrollbars to match the app's color scheme
Fixed the Notes field of Nodons rendering unnecessarily tall sometimes
V1.2 — 6/20/2026
Mobile-first header overhaul, full touch-gesture support, and UI polish across both desktop and mobile.
UI (Desktop)
Zoom control changed from +/− buttons to a single value field, with scroll-wheel support like the Speed field
Header now wraps into additional rows instead of squishing its controls when the window gets narrow
UI (Desktop and Mobile)
Nodon parameters stay faintly visible behind the live values shown in Simulate mode, instead of disappearing entirely
Sidebar is now collapsible
Header bottom border, status bar top border, and sidebar left border now use the accent color
The port you're connecting from now glows with a solid color (magenta for signal ports, blue for object ports) instead of a faint outline
Separator added between the Title field and the Export button in Build mode
Deselect button in Simulate mode now spans the full width of the sidebar footer, matching Build mode
Build mode sidebar footer now has separate Deselect and Delete buttons
Further compacted Notes field when empty
Language button added next to logo (currently disabled; JA translation coming soon!)
UI (Mobile)
Header now uses a flex-wrap layout that stacks into rows based on available screen space
Hover states are fully disabled on touch devices; buttons only respond to taps
Title label is hidden in Simulate mode, reducing the number of header rows
Sidebar now expands and collapses automatically depending on the action
Zoom buttons removed from the mobile header; replaced with pinch-to-zoom
Mobile-specific control list added
Touch Gestures (Mobile)
Pinch with two fingers on the grid to zoom in/out
Long-press now imitates drag actions on Desktop (dragging Nodons to move, dragging ports to make connections, dragging on the grid to draw freeform cut/selection in Delete/Multi-select Mode)
Drag up/down on the Size or Speed number fields to scrub their value, mirroring the scroll-wheel behavior on Desktop
Phone's back button now does the same thing Esc does on Desktop (cancel current action)
V1.1 — 6/15/2026
Quick update to fix incorrect subframe being displayed during Normal View, and changed subframe numbering to a better convention. Added a few more QoL features with Delete Mode and refactored Multi-select Mode.
Simulation
Normal View now simulates all 16 subframes at once, and only renders the final subframe of each frame
Subframe display now counts 1–16 per frame; f=0 shows sf=0 as the pre-run state
Frame step buttons advance to the next/previous sf=16 boundary rather than exactly 16 ticks
Resuming after stepping forward from f=0 now works correctly
Speed changes take effect immediately without causing tick bursts
Header Controls
Clicking the ArduiNodon UI logo now opens the Changelog (hey, that's me!)
COLS×ROWS replaced by a single SIZE label; scroll wheel adjusts columns and rows
Zoom buttons order swapped to − then +; ZOOM and SPEED labels now match the same font as SIZE and TITLE
sf counter given a fixed display width so the layout no longer shifts as the number cycles
Scroll wheel on the Speed input adjusts by ±0.1
Delete Mode
New Delete button in the toolbar (also Del/⌫ with nothing selected, or Esc / click outside to exit)
Click any Nodon or connection to delete it instantly
Click and drag to draw a non-intersecting cut line which will delete any connection it crosses
Draw a self-intersecting loop to delete all Nodons whose centers fall inside it; the enclosed area fills red once the path crosses itself
Multi-select
Replaced the two-corner rectangle selection with freeform draw mode (S key or Multi-select button)
Click individual Nodons to toggle them in or out of the selection
Draw a self-intersecting loop to select all enclosed Nodons; hold Ctrl to add/remove more Nodons to/from the current selection
Move/Copy preview turns all destination tiles red together (with stronger highlighting) if the placement is invalid
Export / Import
Import sets the grid one cell larger than the content bounds in each dimension
Export now shows warnings when Person Nodons or gyro-related Nodons (Shake, Tilt, If Face Up, Rotation Speed, IR Motion Camera, IR Light, Hand) are present
V1.0 — 6/12/2026
Initial release! Full Nodon editor and simulator with import/export functionality.
